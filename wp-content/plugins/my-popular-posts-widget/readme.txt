== Description ==

A very simple popular posts widget which displays posts by comment count (with featured image).

This is a very simple plugin which displays your posts by comment count when activated at your desired sidebar.
It comes with a setting whether you'd like to display your post's featured image or not and also be to adjust the width and height of it.

No CSS styling has been done and it is up to you to customize it.

Please write a review or rate if you like it! :)

== Installation ==

1. Upload the `my-popular-posts-widget` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
1. Activate the plugin 'My Popular Posts Widget' through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer.
1. A new widget called My Popular Posts will appear in the widgets section.
